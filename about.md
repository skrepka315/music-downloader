# Music Downloader

Download any music for Geometry Dash via direct link!

## How to use
1. On the main menu, click the download button (bottom panel)
2. Enter the **Song ID**—the song number under which it will be saved
3. Enter the **direct link** to the MP3 file
4. Click **Download**
5. The song will be saved and available in levels with this Song ID

## Features
- Download via direct link (MP3, OGG)
- Paste URL from clipboard
- Display progress and file size
- Replace existing songs
